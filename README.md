# Data Mining - Apriori Algorithm
This project implements the Apriori algorithm from scratch in Python to discover frequent itemsets and generate association rules across 5 transaction databases.

## Features:

- Full Apriori implementation including candidate generation, support counting, pruning, and association rule mining
- Brute force implementation for performance comparison
- Configurable minimum support and confidence thresholds
- Benchmarks runtime of both approaches across datasets

## How to run:

1. Place your CSV transaction files in the project directory
2. Run python apriori.py
3. Results and runtime comparisons will print to the console

## Technologies
Python, itertools, csv, time
